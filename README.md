# COMPOSITECALENDARWORKER

## About

Worker service for the compositecalendar app

## Configuration

Set the following environment variables:

* Postgress settings:
  * PGHOST
  * PGUSER
  * PGDATABASE
  * PGPASSWORD
  * PGPORT
* Google api settings:
  * CLIENT_ID
  * CLIENT_SECRET
